# Snowflake Sandbox

A sandbox environment for experimenting with Snowflake.

## Setup

### For Apple ARM Users

To run this project, you need to use a virtual environment:

```bash
cd ~/projects
source snowflake-env/bin/activate
python script.py
```

# Snowflake Sandbox

A sandbox environment for experimenting with Snowflake.

## Setup

### For Apple ARM Users

To run this project, you need to use a virtual environment:

```bash
cd ~/projects
source snowflake-env/bin/activate
python script.py
```

## Summary

Along the way, I completed the following steps:

- Install the Snowflake Python APIs.
- Set up a connection to Snowflake.
- Create a database, schema, and table.
- Retrieve object information.
- Programmatically alter an object.
- Create, suspend, and drop a warehouse.
